Work Experience

Compare Asia Group Software Engineering Intern

Singapore Office

Hong Kong Office

CompareAsiaGroup (CAG) is a financial technology startup based in Southeast Asia that seeks to financially empower individuals via a financial management platform. During my two month internship, I worked on the following projects.

  • SMS Gateway

    Developed an SMS-gateway which enables CAG Marketing to send text messages to clients with incomplete financial applications. Implemented the gateway via JHipster, a scaffolding tool which offers an integration of SpringBoot, Swagger, Postgresql, Maven, and React. The gateway leverages the Twilio API to send sms's and listen for responses via a webhook. If the user responds with certain key words, the webhook triggers an api call which searches the database for the appropriate unsubscribe urls and makes a rest call to an internal microservice to alter the user's subscribe status.

  • FB Messenger Chatbot

    Developed an MVP Facebook Messenger chatbot which utilizes the Wit.ai NLP engine and the Botkit.ai framework to interact with clients purchasing personal loans in Malaysia. Crafted a flow where the chatbot can sift through qualified leads based on personal loan history and capture the client's personal information for future contact. Additionally, recorded a demo of the MVP which marks the first demo to be uploaded to the new CAG Youtube Channel, an initiative aimed at increasing visibility of projects within the company.

  • IMDB React App

    Redesigned CAG's front-end developer interview process by building a two-page react node application which uses React hooks and styled components to simulate the IMDB movie interface via their TMDB api. Thereby, optimized the interview process for time efficiency such that prospective candidates can implement a single additional feature as opposed to building the web app from scratch as a gating task in there interview process.

  • Partner Portal

    Implemented login functionality for the Partner Portal, a digital platform that manages user access to highly secure PII for both internal CAG members and external providers. Through Okta's OAuth 2.0 authorization server and related OIDC provider, enabled multi-factor authentication and implemented the authorization code PKCE grant flow. Developed the POC for the Partner Portal using the Fuse-React template which was then integrated with Jhipster.
